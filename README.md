# 종교 스크립트 작성기 ✝️☪️🕉️

YouTube 종교 콘텐츠 제작을 위한 AI 기반 스크립트 자동 생성 도구입니다.

## 지원 종교
- ✝️ 기독교 (Christian)
- ☪️ 이슬람교 (Islam)
- 🕉️ 불교 (Buddhism)
- 기타 종교

## 주요 기능
- 다양한 스크립트 패턴 선택 (스토리텔링형, 질문 폭격형 등)
- AI 이미지 프롬프트 생성
- 유튜브 후킹 스크립트 자동 생성
- 다양한 아트 스타일 및 분위기 설정

## 시작하기

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

### 빌드
```bash
npm run build
```

## 기술 스택
- React 18
- Vite
- Tailwind CSS
